给你一个整数数组 prices ，其中  prices[i] 表示某支股票第 i 天的价格。

在每一天，你可以决定是否购买和/或出售股票。你在任何时候  最多  只能持有 一股 股票。你也可以先购买，然后在 同一天 出售。

返回 你能获得的 最大 利润  。

示例 1：

输入：prices = [7,1,5,3,6,4]
输出：7
解释：在第 2 天（股票价格 = 1）的时候买入，在第 3 天（股票价格 = 5）的时候卖出, 这笔交易所能获得利润 = 5 - 1 = 4 。
   随后，在第 4 天（股票价格 = 3）的时候买入，在第 5 天（股票价格 = 6）的时候卖出, 这笔交易所能获得利润 = 6 - 3 = 3 。
总利润为 4 + 3 = 7 。
示例 2：

输入：prices = [1,2,3,4,5]
输出：4
解释：在第 1 天（股票价格 = 1）的时候买入，在第 5 天 （股票价格 = 5）的时候卖出, 这笔交易所能获得利润 = 5 - 1 = 4 。
   总利润为 4 。
示例  3：

输入：prices = [7,6,4,3,1]
输出：0
解释：在这种情况下, 交易无法获得正利润，所以不参与交易可以获得最大利润，最大利润为 0 。

提示：

1 <= prices.length <= 3 * 104
0 <= prices[i] <= 104

class Solution:
    def maxProfit(self, prices: List[int]) -> int:
        # 解法一:一次遍历 - 由于只要走高持续卖就会持续有利润,而且假设的是第一天就持股了,第二天比第一天高就有得卖
        # 所以只要第二天的价比第一天高就卖出,只要走高就卖出,走低就买入再卖出
        # 就会形成一种线性的现象-只要第二天比第一天高就卖出去拿利润
        # 所以这道题在现实生活中不会发生-因为谁也无法预测股市走势📈
        profile = 0
        for i in range(1, len(prices)):
            if prices[i] > prices[i-1]:
                profile += prices[i] - prices[i-1]
        return profile
        

class Solution:
    def maxProfit(self, prices: List[int]) -> int:
        """
            解法二:dp[i][0/1] -> dp[i][0]表示第i天手上不持股的最大利润,dp[i][1]表示第i天手上有股票的最大利润
            开辟大小为dp[n][2] - n行2列的二维列表,然后去判断
            所以就两种状态:一、今天手上有股票:选择买入利润减少或不买利润不变. 二、今天手上没有股票:选择卖出昨天的股票增加利润或不卖利润不变
        """
        dp = [[0 for i in range(2)] for i in range(len(prices))]
        print(dp)
        # 第一天默认手上持股
        dp[0][0] = 0
        dp[0][1] = -prices[0]
        for i in range(1, len(prices)):
            # 手上没有股票则利润的最大值就是:前一天手上没有股票,或者昨天手上有股票以今天的股票价格卖出产生的利润
            dp[i][0] = max(dp[i-1][0], dp[i-1][1]+prices[i])
            # 手上有股票的利润的最大值就是:前一天手上没有股票,今天买入以后剩余的利润,以及昨天有股票今天无事发生的利润
            dp[i][1] = max(dp[i-1][0]-prices[i], dp[i-1][1])
        print(dp)
        max_profile = 0
        for i in dp:
            if max(i) > max_profile:
                max_profile = max(i)
        return max_profile
